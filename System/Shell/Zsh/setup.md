# zsh Setup Manual


## Install zsh

ubuntu  

```sh
sudo apt install zsh
```


---


## Change Login-Shell

ubuntu  

```sh
chsh -s $(which zsh)
```


---


## Edit Zsh Config

add & edit `.zshrc`

## Install Plugin Manager

Oh My Zsh  
```sh
sh -c "$(curl -fsSL https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
```
